# Tracking Money

개인 보유종목, 관심종목, 미국 선행 지표, 뉴스 이슈, 캘린더를 한 화면에서 보는 투자 대시보드입니다.

## 주요 기능

- 내일 국내장 방향점수
- 기준/상방/하방 내일장 예측 시나리오
- 장 시작 전 확인 순서, 진입/보류 조건, 집중 심볼을 묶는 장전 운전석
- 현재 KST 기준 야간/장전/개장직후/장중/마감/장후 할 일을 나누는 장중 컨트롤 타워
- 뉴스, 공시, 캘린더를 시간순으로 합쳐 보는 촉매 레이더
- 선행지표, 뉴스, 공시, 캘린더, 알림을 압박/기회 점수로 합치는 시장 충격 레일
- SOX, NQ=F, USD/KRW, VIX, 금리 가정을 직접 바꿔보는 가정 시뮬레이터
- SOX, NQ=F, USD/KRW, VIX 추가 악화 시 내 포트폴리오 충격을 보는 야간 스트레스 테스트
- 시세, 뉴스, 일정, 포트폴리오를 합친 복사용 장전 브리핑
- 시세, 뉴스, 공시, 캘린더 연결 상태를 가중 평가하는 실데이터 신뢰도
- 시세, 뉴스, 공시, 캘린더가 언제 갱신됐는지와 갱신 필요 여부를 보여주는 데이터 최신성 가드
- 데이터 소스별 실데이터/대체 데이터 판정, 호출 경로, 필요한 환경변수 진단
- 보유/관심종목, 뉴스 키워드, 개인 일정, 알림, 투자노트, 실데이터 연결을 합산하는 개인화 완성도
- 예측에 반영된 지표, 뉴스, 공시, 일정, 포트폴리오 기여도를 나눠 보는 예측 근거 감사
- 어떤 변수 변화가 상방/하방 전환을 만드는지 보여주는 예측 민감도·전환조건
- KOSPI/KOSDAQ 실제 흐름과 예측을 비교하는 장후 예측 검증 리포트
- 복기 히스토리 기반 예측 적중률, 실행률, 개선 포인트를 보는 예측 캘리브레이션
- 보유/관심종목별 분할 매수, 축소 감시, 대기 기준을 만드는 내일 실행 계획
- 보유종목과 관심종목 영향도
- 보유 비중, 섹터 노출, 선행지표를 합친 포트폴리오 플레이북
- NQ 선물, SOX, VIX, 달러/원, 미국 금리 등 선행 지표
- NQ 선물, SOX, 환율, VIX를 국내장 예상 범위와 개장 전 체크리스트로 바꾸는 미국 선행지표 브리지
- Yahoo Finance chart 기반 무료 시세 프록시
- 네이버 뉴스 API 기반 사용자 키워드 뉴스 피드
- 실시간 뉴스와 보유/관심종목 민감도를 합친 종목별 이슈 영향판
- 뉴스, 공시, 일정을 이슈별 상방/하방 갈림길로 나누는 이슈 시나리오 매트릭스
- 시세, 뉴스, 공시, 일정, 액션 큐를 종목별로 묶는 종목별 이슈 브리핑
- 보유/관심종목 기반 이벤트 캘린더 API
- OpenDART 기반 국내 공시/실적 원문 체크
- 보유종목/관심종목 브라우저 저장 및 편집
- 개인 실적, 매크로, 종목 이벤트 캘린더 저장 및 액션 큐 반영
- 가격, 등락률, 뉴스 키워드, 방향점수 기반 사용자 알림 규칙과 브라우저 알림 히스토리
- 지표, 뉴스, 캘린더, 관심가를 합친 오늘 액션 큐
- 장전 계획, 리스크 대응, 장후 리뷰를 저장하는 투자 저널
- 예측 검증 결과와 장후 리뷰를 날짜별로 쌓는 복기 히스토리
- 실적, 매크로, 정책 이벤트 캘린더

## 실시간 데이터 연결

앱은 Vercel 서버 함수를 통해 외부 데이터를 가져옵니다. 브라우저가 직접 외부 API를 호출하지 않습니다.

```text
/api/news    네이버 뉴스 API 프록시
/api/quotes  Yahoo Finance chart 기반 시세 프록시
/api/calendar  보유/관심종목 기반 매크로·실적 이벤트 캘린더
/api/disclosures  OpenDART 기반 국내 공시 프록시
/api/health  데이터 소스와 환경변수 연결 상태 진단
/api/profile  선택형 서버 프로필 동기화
```

`/api/quotes`는 아래 기본 심볼을 수집합니다.

```text
005930.KS  삼성전자
000660.KS  SK하이닉스
035420.KS  NAVER
373220.KS  LG에너지솔루션
AAPL       Apple
TSLA       Tesla
NVDA       NVIDIA
KOSPI      KOSPI
KOSDAQ     KOSDAQ
NQ=F       나스닥100 선물
ES=F       S&P500 선물
^SOX       필라델피아 반도체지수
^VIX       변동성 지수
DX-Y.NYB   달러 인덱스
^TNX       미국 10년물 금리
KRW=X      달러/원
```

시세 수집이 실패하면 기존 샘플 데이터로 자동 fallback됩니다.

보유종목, 관심종목, 뉴스 키워드, 사용자 알림 규칙은 화면에서 직접 추가, 수정, 삭제할 수 있습니다. 입력값은 브라우저 `localStorage`에 저장되며, 새로 추가한 국내 6자리 종목과 미국 티커도 `/api/quotes` 수집 대상에 포함됩니다. 뉴스 키워드는 `/api/news` 수집 파라미터와 뉴스 화면 필터에 바로 반영됩니다. 알림 화면에서는 발동 기록을 남기고, 권한을 허용하면 새 조건 발동을 브라우저 알림으로 받을 수 있습니다.

`/api/calendar`는 현재 추적 중인 심볼을 기준으로 국내장 개장 전 선행지표 점검, 주요 매크로 이벤트, 정책 이벤트, 종목별 실적 발표 구간을 생성합니다. `watch`는 매일 확인할 체크리스트, `estimated`는 공식 일정으로 재확인이 필요한 추정 이벤트입니다. 캘린더 화면에서는 개인 실적, FOMC, 환율 이벤트, 종목 메모 일정을 직접 추가할 수 있고, 개인 일정도 예측 근거와 액션 큐에 같이 반영됩니다.

`/api/disclosures`는 OpenDART의 공시검색 API를 서버에서 호출해 국내 보유/관심종목의 최근 공시 원문 링크, 중요도, 예상 영향을 정리합니다. 현재 기본 매핑은 삼성전자, SK하이닉스, NAVER, LG에너지솔루션을 포함합니다.

예측 화면은 방향점수, 선행 지표, 뉴스, DART 공시, 캘린더, 보유종목 비중을 룰 기반으로 합쳐 기준/상방/하방 시나리오, 예상 출발 범위, 핵심 변수, 장전 체크리스트를 보여줍니다. 장전 운전석은 예측, 민감도, 선행지표 브리지, 종목별 이슈 영향판, 액션 큐를 합쳐 08:45부터 09:30까지 확인 순서, 진입 조건, 보류/방어 조건, 집중 심볼을 한 카드로 정리합니다. 장중 컨트롤 타워는 현재 KST 기준으로 야간, 장전, 개장 직후, 장중, 마감, 장후 단계를 나누고 다음 체크포인트, 대응 모드, 우선 작업, 가드레일을 다시 계산합니다. 촉매 레이더는 뉴스, 공시, 캘린더를 즉시/오늘/야간/다가오는 버킷으로 묶어 어떤 이벤트를 먼저 봐야 하는지 정렬합니다. 시장 충격 레일은 촉매 레이더, 선행지표, 사용자 알림을 다시 압박 지수와 기회 지수로 합산해 장전 대응 순서를 보여줍니다. 가정 시뮬레이터는 SOX, NQ=F, USD/KRW, VIX, 금리 변화폭을 직접 바꿔 KOSPI/KOSDAQ 예상 범위, 포트폴리오 추정 영향, 민감 포지션을 즉시 계산합니다. 야간 스트레스 테스트는 SOX, NQ=F, USD/KRW, VIX, 금리가 추가로 악화되는 경우의 KOSPI/KOSDAQ 압박 범위, 포트폴리오 추정 충격, 민감 포지션, 방어 체크리스트를 계산합니다. 실데이터 신뢰도는 시세, 뉴스, 공시, 캘린더 연결 상태를 가중 평가해 현재 예측을 얼마나 강하게 참고해도 되는지 알려주고, 데이터 최신성 가드는 각 소스의 마지막 갱신 시각, 권장 갱신 주기, 갱신 필요 여부를 따로 보여줍니다. 각 소스의 실데이터/대체 데이터 여부, 호출 경로, 필요한 환경변수, 다음 확인 액션도 함께 표시합니다. 예측 근거 감사는 지표, 뉴스, 공시, 일정, 포트폴리오별 기여도와 데이터 품질 경고를 나눠 보여줍니다. 예측 민감도는 현재 방향점수에서 상방 전환과 방어 전환까지 남은 점수, 그 전환을 만들 수 있는 지표/뉴스/보유 리스크, 확인해야 할 트리거를 계산합니다. 미국 선행지표 브리지는 SOX, NQ=F, USD/KRW, VIX, US10Y, DXY를 국내장 관점으로 다시 계산해 KOSPI/KOSDAQ 예상 범위, 장 초반 전략, 우선 확인 심볼, 개장 전 체크리스트를 보여줍니다. 포트폴리오 플레이북은 평가금액, 당일 추정 손익, 국내/미국/섹터 노출, 집중도, 리스크 신호, 종목별 행동 기준을 함께 계산합니다. 내일 실행 계획은 이 플레이북을 바탕으로 신규 노출 한도, 분할 매수/축소 감시 수량, 가격대, 리스크 가드레일을 만듭니다. 장전 브리핑은 이 데이터를 시장 방향, 포트폴리오 판단, 오늘 이슈, 먼저 할 일로 정리하고 클립보드 복사를 지원합니다.

대시보드와 알림 화면의 액션 큐는 방향점수, 선행 지표, 보유종목 비중/등락, 관심가 근접, 네이버 뉴스 중요도, DART 공시, 캘린더 이벤트를 합쳐 오늘 먼저 확인할 항목을 정렬합니다. 종목별 이슈 영향판은 실시간 뉴스가 직접 언급한 종목, 섹터 관련성, 선행지표 민감도를 함께 계산해 보유/관심종목별 뉴스 점수, 예상 움직임, 대응 액션을 보여줍니다. 이슈 시나리오 매트릭스는 뉴스, 공시, 일정을 이슈 단위로 묶고 각 이슈가 상방 재료로 확인될 때와 하방 악재로 확산될 때의 방향점수 변화, 확인 조건, 대응 액션을 나눠 보여줍니다. 종목별 이슈 브리핑은 시세 변화, 뉴스 민감도, 공시, 일정, 액션 큐를 종목 단위로 다시 묶어 오늘의 판단, 주요 촉매, 다음 체크 항목을 보여줍니다. 알림 화면에서는 가격 이상/이하, 등락률 이상/이하, 뉴스 키워드, 방향점수 이상/이하 조건을 직접 만들고 현재 발동 여부를 확인할 수 있습니다.

투자노트 화면에서는 오늘 액션 큐를 체크리스트로 처리하고, 장전 계획/리스크 대응/장후 리뷰를 브라우저에 자동 저장합니다. 예측 검증 리포트는 KOSPI/KOSDAQ 수집값이 있으면 실제 지수 흐름과 방향점수를 비교하고, 없으면 보유종목 가중 흐름으로 대체해 장후 리뷰 초안을 만듭니다. 예측 캘리브레이션은 저장된 복기 히스토리와 오늘 검증 리포트를 합쳐 평균 검증 점수, 적중률, 액션 실행률, 다음 보정 포인트를 계산합니다. 작성한 저널은 복기 히스토리에 보관해 날짜별 실행률, 예측 판정, 장전 계획, 장후 리뷰를 다시 볼 수 있습니다. 날짜가 바뀌면 새 저널이 시작되고, 과거 날짜 저널은 가능한 경우 복기 히스토리로 보존됩니다.

설정 화면의 `실데이터 수신 상태`와 `서버 연결 진단`에서는 현재 화면이 실제로 받은 시세/뉴스/공시/캘린더 상태, 호출 중인 서버 함수 경로, 필요한 Vercel 환경변수, 환경변수 설정 여부, 아직 예정 단계인 데이터 소스를 확인할 수 있습니다. `개인화 완성도`는 보유/관심종목, 뉴스 키워드, 개인 일정, 알림, 투자노트, 실데이터 연결이 예측과 액션 큐에 얼마나 채워져 있는지 점수화하고 다음 보강 액션을 보여줍니다. `뉴스 키워드 관리`와 `데이터 백업`에서는 개인 뉴스 키워드, 알림 규칙, 알림 기록까지 JSON으로 백업/복원할 수 있습니다.

`서버 동기화`는 Upstash Redis REST 호환 저장소를 선택적으로 사용합니다. 환경변수를 설정하면 보유종목, 관심종목, 개인 일정, 뉴스 키워드, 알림 규칙, 알림 기록, 투자노트와 복기 히스토리를 서버에 저장하고 다른 기기에서 불러올 수 있습니다. 동기화 키는 화면에서 입력한 요청에만 사용하고 브라우저에 저장하지 않습니다.

## API 환경변수

Vercel 프로젝트의 `Settings > Environment Variables`에 아래 값을 추가합니다.

```text
NAVER_CLIENT_ID=네이버_개발자센터_Client_ID
NAVER_CLIENT_SECRET=네이버_개발자센터_Client_Secret
OPENDART_API_KEY=오픈다트_API_인증키
KV_REST_API_URL=Upstash_Redis_REST_URL
```
